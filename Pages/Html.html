NECESSARIO

Non esiste un solo modo o un solo programma per creare paging web, nel nostro caso utilizzaremo visual studio code, un programma molto utilizzato anche per le sue numerose estensioni e la possibilita di installarlo su numerosi sistemi operativi (macos windows, linux, android)

PROVARE LE PAGINE

Per creade un a pagina html é necessario creare una cartella, e creare dentro 3 cartelle (chiamate js, img, css) e un file chiamato index.html.
Adesso aprire visual studio e selezionare "open folder" (apri carmella) e selezionare la cartella del progetto.
Adesso selezionare il file index.html è scrivere html: 5 (apparirà al completamento automatico) e salvare, adesso se forniamo nella nostra cartella, vediamo che l'icona del file e cambiata e cliccandoci sopraci aprira la pagina del brownser,per evitare e di dover sempre aggiornare la pagina per vedere i cambiamenti apportati possiamo aggiungere a visual studio code l'estensione "live server".
Una volta installata ci comparită in basso la scritta GOLIVE cliccandla (utilizzabile solo se si apre la cartella come in precedenza) si aprira il browser.
Adesso ogni volta che apporteremo una modifica alla pagina tramme visual studio code, la pagina si aggiornera in automatico mostrando subito le modifiche.

UNA STRUTTURA A SCATOLE

Per creare la nostra pagina web andiamo ad inserire dentro al tag body tutti i tag che anremo ad utilizzare.il primo tag che andiamo a vedere e il div.
il "div" possiamo considerarlo come una scatola o un contentore, al suo interno possiamo inserire sia altri div che altre scatole e cosi via.
<div>
</div>

LINK

All'interno della pagina web possiamo mettere del rimandi ad altre pagine web o ad una parte di pagina, questo tramme il tag a.
Questo tag. possiede un attributo href="", all'interno delle virgolette dobbiamo inserire il percorso della pagina da collegare oppure l'id del tag a cui andare.per la differenza tra id class consultare la pagina sul css.
<a href=""> </a>

IMMAGINI

Nella pagina possiamo aggiugere delle immagini tramite il tag img.come per i link nell'attributo href inseriamo il link dell' immagine.
Per visualizzare l'mmagine anche offline è possibile scaricare immagine e inserirla nella cartella img. in questo caso inseriamo il percorso relativo della foto.
< img alt="" src="" >

PARAGRAFI

Per non lasciare tutto il testo sparso in un div è possibile raggrupparlo in paragrafi.questo inserendo il testo all'interno di un tag p.
<p>
</p>

CASELLE DI TESTO

Per poter avere un "dialogo" tra l'utente e il sito web e possibile inserire dei tag definiti moduli.
Il primo modulo di cui parliamo è la casella di testo, esistono diversi tipi di caselle di testo.
< input type="text" >
< input type="number" >
< input type="password" >

PULSANTI

Tramme il pulsante possiamo richiamare una funzione javascript tramite l'evento onclik (approfondimento nella pagina di javascript)
< input type="button" >

< /button >

TENDINA

Un altro modulo molto utilizzato è il menù a tendina, utile per inserire più opzioni in poco spazio.per crearla dobbiamo creare un tac select e all'interno inseramo tanti tag option quante sono le opzioni da inserire all'interno della tendina.
<select>
<option> </option>
<option> </option>
</select>

RADIO & CHECK BOX

I radio e check bok vengono utilizzati per le domande a scelta multipla.per collecare tra di loro i tag radio o checkbox basta dargli e la stessa classe.
i tag radio permettono 1 sola scelta, mentre i checkbox permettono di se sceglire più di una opzione.
---Radio---
< input type="radio" class="prova" value="1" >
< input type="radio" class="prova" value="2" >
< input type="radio" class="prova" value="3" >
---check box---
< input type="checkbox" class="prova" value="1" >
< input type="checkbox" class="prova" value="2" >
< input type="checkbox" class="prova" value="3" >
