COLLEGARE CSS E HTML

Per fare in modo che le modifiche vengano apportate all'interno dell'html dobbiand collegame i 2 file (è possibile non utilizzare un file ma creare le regole direttamente nell'htme) possiamo collecare sia un file locale che abbiamo sul nostro computer, nella cartella css sia un file esterno.
<link rel="stylesheet" href="">

ID CLASS

Nel css per apportare le "modifiche" ad un "tag" dobbiamo prima identificarlo.
possiamo selezionare tutti i tag di quel tipo, selezionare quelli con un determinato attributo oppure utilizzare un id per identificare un singolo tag in tutta la pagina oppure la class per selezionarne più di uno (anche di tipo diverso)
<div id="ID" class="classe" >
</div>

DIMENSIONI

Tramite il Css possiamo modificare le dimansioni dei tag sia per larghezza che per altezza, variando anche le loro parti interne come margin, padding e border.
Gli elementi possono essere modicati si completamente, sia variando solo 1 dei 4 lati.
Il border inoltre si può anche randere invisibile o variarne il colore.
Le misure possono essere scritte in deiverse unità di misura:%, px,...
width 20px;
height 30px;

SFONDO

I css permette di modificare lo sfondo dei tag come il div.
Con background-color si imposta un colore, mentre con background-image si puo inserire un immagine sia locale che online.
Come sfondo nel caso dell'immagine si ripeta.
È possible inserire il comanod background-repeat: no repeat per impedirlo.
background-color: #000000;
background-image: url("")

TESTO

È possible variare il colore, font e dimensione tramite i comandi color, font-family & fony-size.
color: #346578;
font-family: 'Times New Roman'
fony-size: 20px;

ALLINEAMENTO

Tramine il css é possibile vamare callineamento, prima rendendo il tag di tipo flex, poi e possibile alunearli in altezza e larghezza.
display: flex;
justify-content: center;
align-items: center;

Per integrare bootstrap nella propria pagina web è sufficente aggiungere un file css e un file js, ricordandosi che javascript bisogna inserirlo al fondo della sezione body.

RIGHE E COLONNE

Con bootstrap possiano rendere le pagine responsive quindi lgi elementi all interno si adattano alla dimensione dello schermo. per farlo bisogna dividere la pagina in righe e colonne.
Per farlo aggiungiamo la classe row per le righe (sempre di tipo flex), all interno possimo creare le colonne inserendo la classe col in una riga ci stanno 12 colonne, poi le colonne andranno a capo.
Per modificare la larghezza delle colonne e possibie inserire come class al posto di col col-numero- per dare on larghezza. la larghezza 12 occupa tutta la riga, quindi una seconda colonna andrebbe a capo.
È possiblele variare il numero di colonne in base 1 allo schermo (es. smartphon 1 colonna, tablet 2 colonne, computer 3colonne) aggiungendo una sigla tra una colonna e la dimensione.
<div class="row" >
<div class="col" >
</div>
</div>

ALLINEARE GLI ELEMENTI

Con boostrap possiamo modificare l'allineamento delgli elementi. come facciamo con le regole css, ma usando classi già integrate nella libreria, anche per rendere i div di tipo flex o block.
<div class="d-flex align-items-center justify-content-center" >
</div>

COLORI

Con bootstrap abbiano accesso a classi che ci permettono di cambiare il colore degli elementi, ovviamente i colori sono limimate, quindi nel caso di un colore specifico e necessario creare una regola css.
<div class="color-primary" >
</div>

MARGIN/PADDING/BORDER

Con bootstrap possiamo anche modificare i margini, bordi e padding, regolandoli con una dimensione da o a 5, possiamo anche cambiare il colore del bordi e renderli visibili o meno
<div class="m-2 b-2 p-2" >
</div>
