NECESSARIO

Il necessario per creare degli degli script in bash e molto semplice e veloce: un sistema operativo linux ed un editor dei testo qualunque che permetta di editare i file sh (estensione degli scrit per linux)

ESEGUIRE FILE

I file in bash sono molto semplici da eseguire, l'unica cosa importante è aver dato i permessi al file per essere eseguito( da fare solo la prima volta che si esegue il file) e che la prima riga indichi che è uno script con il path #!/bin/bash
chmod +x nomefile
./nomefile

COMANDI DI BASE

Negli script possiamo inserire una serie di comandi che eseguiremmo manualmente nel terminale.
rm nomecartella/nomefile -->elimina un file o una cartella
mkdir nomecartella -->crea una cartella
cat nomefile -->stampa il contenuto del file
etc.

VARIABILI

Per dichiarare una variabile basta scrivere il nome, assegnandoli subito un valore.
bisogna ricordarsi che per usarla bisogna mettere prima del nome il simbolo $
In bash si posso creare anche dei vettori, ovvero una serie di variabili a cui si accede mediante un indice
n1=14 #creao una varibile e gli inserisco il valore di 14
echo $n1 #stampo a video il valore della variabile
echo "il volore della variabile è : $n1" #stampo un testo fisso con alla fine la variaibile

numeri=() #dichiaro un vettore
read numeri[0] #chiedo in input un valore e lo salvo nella prima cella del vettore
echo -n "${numeri[0]}" #stampo la prima cella del vettore

INPUT DA TASTIERA

Per catturare un valore messo in unput da un utente utilizzo il comando read seguito dalla variabile in cui salvare quanto inserito dall'utente
echo -n "inserisci un numero" #con -n indico che non deve andare a capo
read numero

IF

Nell' if la condizione deve essere messa tra le doppie parentesi quadre, e al posto di usare le normali condizione si utilizzano una coppia di carattteri.
if [[ $variavile -gt 0 ]]
then
*comandi*
else
*comandi*
fi

-gt --> >
-lt --> <
-eq --> ==
-ge --> >=
-le --> <=

WHILE E FOR

Il for al differenza di if e while al posto di avere le doppie parentesi quadre vuole le tonde, mentre il while si comporta come l'if.
La cosa importante è mettere all'inizio il do e alla fine il done
for (( i=0;i< 5;i++))
do
*comandi*
done

while [[ condizione ]]
do
*comandi*
done

FUNZIONI

Per facilitare e ordinare lo script possimo creare delle funzioni, che poi andiamo ed ogni volta che ci servono, la cosa importante è che le vunzioni devono essere dichiarate all'inizione dello script.
Per richiamere una funzione è sufficiente scrivere il nome della funzione da richiamere
#!/bin/bash
maggiore() {
*comandi* }
#resto dello script
echo "inizio script"
maggiore
echo "resto dello script"
