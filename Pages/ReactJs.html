INCLUDERE REACT

React è una libreria di Javascript che permette di creare delle web-app, ovvero delle applicazioni che funzionano all'interno del nostro browser.
Un altra caratteristica di react è che la pagina non si ricarica mai, avitando di aspettre un caricamento tra un una pagina e un altra.
Per utilizzare react è necessario avere l' NPM installato sul proprio pc, reperibile installando NodeJs, successivamente aprire un terminale nella cartella deve si vuole lavorare e digitare il comando per creare una app in react.
Inoltre è consigliabile installare l'estensione di reactJs per Visual studio code
npm create-react-app nomeCartella

PACKAGE.JSON

Il file package.json è un file molto importante per in nostro progetto,
al suo interno sono scritte tutti i moduli che andiamo a utilizzare nel nostro progetto.
questi moduli sono all'interno della cartella node_modules, possimo anche scaricarne di altri l'unica cosa a cui bisogna prestare attenzione sono le dipendenze.
Infatti ogni modulo ha le proprie dipendenze, che andrà a scaricare quando scaricachiamo il modulo stesso.
Per vedere le dipendenze basta aprire il teminae nela cartella del progetto e digitare:
npm ls -a

PER INIZIARE

In react per creare pagine web lavoro all'interno del file app.js, all'interno della cartella src.
All'inizio è già presente del codice, ma noi cancelliamo l'intero contenuto del file e, usando l'esensione di react per VSCode scriviamo il comando per creare il nostro componente:
racfe

CREARE PAGINE

In react per non si utilizza il linguaggio HTML, ma il linguaggio JSX ovvero un incrocio tra javascript e HTML.
Per creare una pagina è sufficiente scrivere il codice HTML all'interno del return, ci sono alcuni cambiamenti dall' HTML, principalemente l'attributo class che in react si chiama ClassName, e per il fatto che la return ritorna solo 1 div, quindi è necessario contenere tutti gli altri tag all'interno di un div padre.
return(
<div ClassName="classe" >

</div>
)

PROVARE PAGINE

Per provare le pagine in react e necessario di nuovo usare l'npm, questa volta besta aprire un terminale nella cartella del progetto e digitare il seguente comando:
npm start

IMPORTARE IL CSS

Per utilizzare il css è necessario importarlo nel nell'app.js mediante il comndo import, subito all'inizio del file
import ./App.css

JAVASCRIPT

Per creare delle funzioni javascript in react basta creare dell Arrow-Function all'interno del componente ma prima del retrun
function App() {
const Funzione=()=>{
console.log("ciao") }
return(
<div ClassName="classe" > </div>
) }

USESTATE

In react per stampare variabili non è possibile usare quaryselector o getelementbyid, ma è necessario usare gli stati, che permettono di modificare la pagina senza aggiornarla
Questo Hook ci permette di creare una variabile prima e dopo una "funzione" da usare per variare il contenuto del'hook.
Quando definiamo lo stato possiamo inizializarlo a un valore che possimo scrivere tra le parentesi dopo "useState"
import React,{UseState} from 'react'
function App() {
const [stato,setstato]=useState("")
const Funzione=()=>{
setstato("Ciao") }
return(
<div ClassName="classe" >
{Stato}
</div>
) }

USEREF

Mettiamo il caso che dobbiamo assegnare un valore a un tag, oppure dobbiamo combiarne un attributo.
In react non possimo nè usare quaryselector nè getelementbyid, ma, per ovviare al problema, è presente UseRef che salva in usa variabile tutti i dati di un tag, quindi possiamo accederci, indicando che vogliamo il tag corrente con current e successivamente l'azione da fare come SetAttribute.
import React,{UseRef} from 'react'
function App() {
const cambiacolore=useRef()
const Funzione=()=>{
cambiacolore.current.setAttribute("style","background-color:#ffffff) }
return(
<div ClassName="classe" ref={cambiacolore} >
</div>
) }
