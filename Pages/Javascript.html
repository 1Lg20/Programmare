NECESSARIO

Per programmare in javascript non sono necessarie molte cose, basta un semplice editor di testo.

COLLEGARE HTML E JAVASCRIPT

Collegare html e js e molto semplice, basta usare il tag script nel html posigionandolo dovunque anche all'interno del tag body.
Possiamo scrivere codice javascript sia all'interno del html, dentro al tag script, opputre su un file js a parte e collegarlo con l'attributo src del tag script
< script src=""> < /script>

FUNZIONI

I comandi javascript vengono eseguiti appena il browser arriva al tag script, quindi evitare che vengano eseguite parti di codice che modificano tag non ancora cariccati il codice si deve suddividere il funzioni.
const funzione=()=>{

}

EVENTI

Per mandare in esecuzione le varie parti di codice possiamo usare degli eventi per richiamare delle funzioni.
Questi vanno messi come attributo dei tag html e richiameranno la funzione di javascript corrispondente.
Ecco un esempio di eventi più comuni:
<div onload="funzione()" > </div> -->Richiama una funzione quando il tag è completamente caricato, lui e tag al suo interno
<div onclick="funzione()" > </div> -->Richiama una funzione quando si clicca sul tag
<div onchange="funzione()" > </div> -->Richiama una funzione quando cambia il valore del tag, utile per le caselle di testo.
<div onchange="()=>{}" > </div> -->se si vuole si può creare una funzione direttamente all'interno del tag.

CONSOLE LOG

Uno dei etodi più demplici per stampare a video qualcosa è utilizzare il comando console.log() che permette di stapare un valore, stringa o variabile nella console, a cui si può accedere facendo tasto destro analizza/ispeziona-->console
console.log("ciao")

VARIABILI E VETTORI

Per salvare dei dati durante l'esecuzione del programma possiamo crare della variabili, ricordando che javascript non è tipizato, quindi non dobbiamo dichiarare il tipo della variabile
let variabile //uso let per dire che creo una variabile
console.log(variabile)

WHILE E FOR

Per eseguire più azioni uguali di seguito(es. scorrere un vettore) per non ripetere il codice possiamo creare un ciclo for, metre per controllare una variabile possiamo usare un ciclo while
while(condizione){
}

for(i=0; i< LenVettore; i++){
console.log(vettore[i])
}

QUERYSELECTOR

In javascript possiamo interagire con l'html, andando a usare document.queryselector() per selezionare un tag html usando l'id o la classe.
Nel caso la classe si ripeta più volte verra preso in considerazione solo il primo tag per selezionarli tutti usare queryselectorall.
let tag=document.queryselector("#id")

SETATTRIBUTE & InnerHTMl

Una volta selezionato il nostro tag possiamo eseguire una serie di operazioni, come cambiargli un attributo o settare una scritta al suo interno, usando rispettivamente setattribute o innerHTML
let tag=document.queryselector("#id")
tag.setattribute("href","https://www.google.com") -->imposto al tag html un attributo href con il collegamento a sito esterno
tag.innerHTML="ciao" -->imposto all'interno del tag la scritta ciao

CREARE UN TAG

Con javascript possiamo anche creare un tag dal nulla e metterlo nell'html usando il funzione createElement, poi lo appendiamo ad un tag già presente nel file html.
Ovviamente al nuovo tag possiamo assegnare attributi o settare un contenuto, oppure a sua volta usarlo come contenitore per creare altri tag.
let nuovoTag=document.createElement("div")
tag.appendChild(nuovoTag)
